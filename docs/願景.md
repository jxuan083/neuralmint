# NeuralMint — 前景與發展願景

---

## 我們在解決什麼問題

全球 AI API 市場正在高速成長，但有一個根本性的矛盾沒有被解決：

**所有算力都掌握在少數幾家公司手裡。**

OpenAI、Anthropic、Google 可以在任何時候：
- 封鎖你的帳號
- 限制你所在的地區
- 配合政府要求下架模型
- 單方面漲價

這不是假設，而是已經發生的事。

NeuralMint 的答案是：**把 AI 算力還給個人，讓任何人都能參與提供和使用。**

---

## 運作邏輯

任何人都可以把自己電腦的 GPU 閒置時間貢獻出來，跑 AI 推論任務，賺取 NMB token。

任何開發者都可以用 NMB 支付，透過一個 OpenAI 相容的 API 使用這些算力——不需要信用卡、不需要帳號審核、不會被地區限制。

礦工和開發者之間的媒合、付款、獎勵分配，全部由智慧合約自動執行，沒有中間人。

```
開發者付 NMB → 礦工跑推論 → 智慧合約自動分配獎勵
                     ↑
            全球任何人都可以是礦工
```

---

## Token 經濟設計

NMB 的總供給量固定為 2,100 萬枚，不可增發。

每完成 10,000 次 AI 任務，礦工獎勵減半（50 → 25 → 12.5 NMB），這個機制與比特幣相同，確保早期參與者獲得最高回報，同時隨規模成長形成自然的稀缺性。

平台內建 DEX，礦工賺到的 NMB 可以直接在鏈上兌換 MATIC，不需要依賴中心化交易所。

---

## DAO 治理

NeuralMint 沒有 CEO，也不會有。

持有 NMB 的人共同決定平台的走向：哪些模型可以上架、礦工質押門檻要設多少、獎勵比例怎麼分配。每一個決策都透過鏈上投票執行，結果由智慧合約自動落地，沒有任何人可以單方面更改。

這代表 NeuralMint 從第一天起就不是一間公司的產品，而是一個社群共有的基礎設施。

---

## 市場機會

### 被忽略的用戶

全球有一大批開發者無法使用主流 AI API 服務：

- **東南亞、中東、非洲**：沒有國際信用卡，或 OpenAI 不支援所在地區
- **隱私敏感應用**：醫療、法律、新聞媒體，不希望資料流經美國伺服器
- **Web3 原生應用**：已經有錢包，crypto 支付是自然的選擇

這些用戶加起來保守估計超過 5,000 萬人，對應年化市場規模約 **60 億美元**，且幾乎沒有直接競爭者。

### 成本優勢

電費低廉地區（越南、哈薩克、東歐）的礦工，推論成本遠低於 AWS 數據中心。隨著礦工網路擴大，NeuralMint 可以提供比中心化服務低 30–50% 的定價，同時礦工仍有利潤。

## 規模願景

| 階段 | 礦工數 | 日均 API 呼叫 | 里程碑 |
|------|--------|--------------|--------|
| 冷啟動完成 | 500 | 1 萬次 | NMB 形成真實市價，飛輪啟動 |
| 成長期 | 1 萬 | 50 萬次 | 比中心化便宜 40%，非 crypto 用戶開始採用 |
| 規模化 | 10 萬 | 1,000 萬次 | 企業級採用，DAO 治理活躍 |
| 成熟生態 | **100 萬** | 1 億次以上 | 算力超越單一雲端供應商，物理上無法被關閉 |

**百萬礦工代表什麼？**

全球閒置 GPU 數量超過 5 億台，100 萬礦工只需滲透其中 0.2%。在這個規模下，NeuralMint 的等效算力超越任何現有 AI 雲端服務，延遲媲美中心化 API，成本更低，且沒有任何單點可以被關閉。

這不再只是一個 AI API 平台，而是一層**不可審查的 AI 基礎設施**。

---

## 為什麼現在是對的時間點

1. **AI 需求爆炸**：開發者對 LLM API 的需求在過去兩年成長超過十倍，且仍在加速。

2. **中心化風險上升**：各國政府開始對 AI 公司施壓，要求審查模型輸出、保留用戶資料。中心化 AI 的合規成本和政治風險正在升高。

3. **硬體普及**：消費級 GPU 的性能已經足以跑 7B–13B 模型，閒置算力的潛在供給從未如此充足。

4. **OpenAI 相容性**：不需要說服任何人換工具。改一行 `base_url`，現有應用立刻可以跑在 NeuralMint 上。

---

## 唯一的挑戰：冷啟動

NeuralMint 的技術架構、token 經濟、DAO 治理已經完整。

真正的挑戰只有一個：**在飛輪自行轉動之前，需要有人願意撐住。**

歷史上，每一個成功的去中心化網路都經歷過這段時期。以太坊、Bittensor、Uniswap——它們的早期參與者，都是在「還沒有人用」的時候進場的人。

撐過去之後，剩下的就是飛輪。
